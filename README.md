# mypkg

このリポジトリは未来ロボティクス学科のロボットシステム学の授業の課題で作ったものをあげるためのものです。

# インストール手順
コマンドを使用するには、リポジトリをクローンしたあと、robosys2022へ移動してください。

```
$ git clone https://github.com/Nao107010/robosys2022
$ cd robosys2022
```

# 動作確認済み環境
以下の環境で動作確認がされています。
* OS Windows10
* Ubuntu 20.04
